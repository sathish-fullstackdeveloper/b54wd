Welcome to the CRUD DEMO application @ Guvi.in which demonstrates the CRUD operations for a user management system.

The following are the endpoints from the mockapi.io API which is used to demonstrate the CRUD operations:

Endpoint Type Description
/users GET get all users
/users/:id GET get user by id
/users POST create new user
/users/:id PUT replace user by id
/users/:id PATCH update user by id
/users/:id DELETE delete user by id

To perform the CRUD, please use the navigation bar above.

  • View Users - to view all the users
  • Create a New User - to create a new user
  • Edit / Update a User - to edit / update a user
  • Delete a User - to delete a user
