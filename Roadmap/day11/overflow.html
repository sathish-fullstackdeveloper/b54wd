Content that overflows will create a scrollbar. This is the default behavior.
  • Changing the overflow value to hidden will hide the content that overflows.
  • Changing the overflow value to scroll will create a scrollbar even if there is no content that overflows.
  • Changing the overflow value to auto will create a scrollbar only if there is content that overflows.
The auto property is similar to scroll, but it only adds scrollbars when necessary. The visible value is the default. It shows the content that overflows the element.
